WonderPushDelegate Protocol Reference

Conforms to NSObject
Declared in WonderPush.h

Overview

The WonderPushDelegate protocol lets you customize various aspects of the WonderPush behavior at runtime.

– wonderPushWillOpenURL:

Lets you overwrite URLs that WonderPush will open using UIApplication:openURL:. (Deprecated: If wonderPushWillOpenURL:withCompletionHandler: is implemented, it will be called instead of this one.)

- (nullable NSURL *)wonderPushWillOpenURL:(NSURL *)URL

Parameters

URL

The URL that WonderPush is about to open.

Return Value

A URL to open, or nil to avoid opening anything. Just return the value of the URL parameter to use the default WonderPush behavior.

Discussion

Lets you overwrite URLs that WonderPush will open using UIApplication:openURL:.
